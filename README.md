# minecraft-react
this is a tiny minecraft clone built with react,vite, threejs, and zustand.

using the three hooks built-in in its library

Controls:

* <kbd>alt + click</kbd> : delete block
* <kbd>1</kbd> : grass
* <kbd>2</kbd> : log
* <kbd>3</kbd> : glass
* <kbd>4</kbd> : wood
* <kbd>5</kbd> : dirt
* <kbd>w</kbd>
<kbd>a</kbd> <kbd>s</kbd> <kbd>w</kbd> : movement
